# kaggle-house-prices
Predicting house prices using machine learning with Kaggle dataset. Implemented Ridge regression with preprocessing.  Outputs were generated in Kaggle environment.

## Project Structure
kaggle-house-prices/
├── data/
├── house-price-github.ipynb
├── README.md
└── LICENSE

## What I did
- Handled missing values using imputation
- Applied feature scaling (StandardScaler)
- Built a Ridge regression model
- Evaluated performance using RMSE

## Result
- Public leaderboard score: 0.14994

## Future Improvements
- Feature engineering (e.g. TotalSF, TotalBath)
- Try ElasticNet and cross-validation

## How to Run
1. Put `train.csv` and `test.csv` in the `data/` folder
2. Open `house-price-github.ipynb`
3. Run all cells
